Git pull request

In this lab we will learn how to use git pull request


Lab Goals:

1. Create a developer with read-only access
2. Fork the project, edit it, do the pull request
3. Merge the changes


== STEP 1)  Create a developer role with read-only access

	Do this on GitHub


== STEP 2) Fork the project

	Fork the project on GitHub
	Edit the forked project
	Commit the changes and issue the pull request
	

== STEP 3) Accept and merge the pull request changes

	You will receive an email from GitHub
	Follow the instructions in this email

Follow these instruction: https://help.github.com/articles/using-pull-requests/
